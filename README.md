# University Psychological Testing System

Веб-приложение для психологического тестирования студентов, разработанное на Django.

## Возможности системы

### Роли пользователей
- Администратор
- Психолог
- Студент

### Администратор
- Управление пользователями
- Смена ролей
- Сброс пароля
- Блокировка и разблокировка пользователей
- Просмотр результатов тестов
- Статистика и графики

### Психолог
- Создание тестов
- Добавление вопросов
- Добавление вариантов ответов
- Редактирование тестов
- Просмотр результатов студентов
- Статистика

### Студент
- Просмотр доступных тестов
- Прохождение тестов
- Результаты видны только администратору и психологу

### Система результатов
По результатам теста определяется уровень:
- Green — низкий уровень риска
- Yellow — средний уровень
- Red — высокий уровень

## Технологии

- Django
- SQLite
- Bootstrap
- Chart.js
- HTML / CSS
